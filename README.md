# json-to-table

JSONをTable化する関数

## 使用ライブラリ

jQuery

## 使用法

json-to-table.js を読み込みます。

>  <script src="json-to-table.js"></script>

json2table を呼び出します。
引数でオブジェクトを渡すとテーブルのjQueryオブジェクトを返します。

> $table = json2table(json);
